SSH 工作流程

页面提交表单，从 struts 的配置文件中找到响应的 action，从 spring 容器中获取由 action 后所指定的 class 的实例，调用 action 后所指定的 method 处理提交的数据与数据库进行交互，由 action 中的 result 接收被调用 method 的返回值转向不同的网页。

struts 在 MVC 框架中充当控制器的角色，使用Struts的目的是为了减少在运用 MVC 设计模型来开发 Web 应用的时间。
spring 使用 spring 作容器，为我们创建及管理 java bean，我们不需要关注实例的生命周期以及实例的创建，需要的时候到容器中去取来即可。
hibernate 连接数据库的框架，提供了按对象操作数据库的功能，我们只要按照格式写好 java 类与数据库中表的映射关系即可使用。
